# 删除网络相机代码总结

## 📋 概述

本次修改删除了项目中所有与网络相机相关的代码，简化系统架构，使项目只支持 Sapera SDK 相机。

**修改日期**：2026-05-18  
**修改原因**：网络相机功能已被砍掉，只保留 Sapera 相机支持  
**影响范围**：3 个核心文件

---

### 2. `ui/CameraStatusBar.py` - 相机状态栏

## 🎯 影响分析

### 正面影响
1. **代码简化**：删除了约 165 行不需要的代码
2. **类型安全**：所有类型注解都明确指向 `SaperaCameraInfo`
3. **逻辑清晰**：不再需要判断相机类型，减少了分支逻辑
4. **维护性提升**：代码更容易理解和维护

### 潜在风险
1. **旧方案文件**：如果旧方案文件中保存了网络相机信息，加载时可能无法匹配到实际相机
   - **缓解措施**：保留了 `port` 字段和方案文件解析逻辑，确保基本兼容性

2. **API 变更**：`switch_camera` 等方法的参数类型从 `Union` 变为 `SaperaCameraInfo`
   - **影响范围**：只影响直接调用这些方法的代码
   - **缓解措施**：项目中所有调用处都已更新
